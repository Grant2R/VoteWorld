# Whereabouts

I want to create a website where people can select what country they are from and they are able to compete against other countries to have the most people submit their country.

## How it works
- Select your country from the dropdown
- Click "Add my country" to cast your vote
- See how your country ranks against others
- Votes are saved so results are always up to date

## Built with
- HTML
- Tailwind CSS
- JavaScript
- localStorage
